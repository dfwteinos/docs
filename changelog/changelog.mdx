<Update label="May 6, 2026" tags={["Interviews"]}>
  ### Interviews

  Lope now supports interview joining bots, including *Google Meetings* and *Microsoft Teams* . Shortly after the interview, our bot transcribes the interview and the user can ask questions based on the interview.
  
</Update>

<Update label="April 22, 2026" tags={["AI Scout"]}>
  ### External & Mixed Search

  Lope now supports **External** and **Mixed** Search. External search, can scrape 800M+ LinkedIn profiles, and based on Lope's algorithm, returns back the best matched-profiles. Mixed search compares candidate both from LinkedIn and internal database. 

  <Info>
    External search supports up to 20 new profiles each time. For better limits, please contact support@withlope.com
  </Info>
  
</Update>

<Update label="April 8, 2026" tags={["Shortlists"]}>
  ### Shortlists

  Candidates can now be added into shortlists for easier management. Pick your champion-candidates, create a new shortlist and add them instantly there.
  
</Update>

<Update label="March 28, 2026" tags={["Candidate Tracking"]}>
  ### Candidate Tracking

  Implemented the Candidate Tracking Feature. Recruiters can now track real-time updates from candidates which are imported from LinkedIn. Changes include:

  1. **Company moves**: When a candidate joins a new company.
  2. **New certifications**: AWS, GCP, PMP and similar.
  3. **New skills**: Skills added to the LinkedIn profile.
  4. **Headline changes**: The title or tagline on LinkedIn.
  5. **Promotions**: Role-level changes inside the same company.
  6. **New awards**: Recognitions and public awards.
  7. **New education**: Degrees and program completions.
  8. **New Projects**: Projects the candidate publishes.

  <Info>
    More fields are coming soon. If you have any custom inquire, please contact the team: support@withlope.com
  </Info>
  
</Update>



<Update label="March 22, 2026" tags={["Candidate Sheet"]}>
  ### Candidate Sheet

  Added the candidate sheet view, a centralized place where every recruiter can see exactly all the available information from a candidate, including:

  1. **LinkedIn** Name, Location, Experiences, Education, Skills.
  2. **Github** profile, including public repositories, total commits, languages.
  3. **CVs** information, with edit capabilities.
</Update>

<Update label="March 5, 2026" tags={["AI Scout"]}>
  ### Internal Candidate Search v.01

  First version of AI Scout is available. Users can import candidates from LinkedIn and search based on:

  1. Current **Job Position Title**.
  2. **Skills** they have on the candidate's LinkedIn Profile.
  3. **Location** mentioned at the top of their profile.
  4. **Industry** that their current company is in (along with external information using third party data provider).
  5. **Total Years of Working**, which is industry or job position agnostic. Uses sum of their working experience in the overall calculation.

  ---

  <Info>
    The search right now is **evenly weighted.** That means that each term contributes the same (20/100) in the overall scoring.
  </Info>
</Update>
