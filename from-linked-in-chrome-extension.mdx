---
title: 'From LinkedIn (Chrome Extension)'
---


Add candidates directly from LinkedIn using the **Lope LinkedIn Recruiter** Chrome extension. The extension adds a side panel next to LinkedIn profiles and lets you add candidates with one click, with optional LinkedIn enrichment.

### Install the extension

1. Install the extension from the [Chrome Web Store](https://chrome.google.com/webstore) (search for "Lope LinkedIn Recruiter") or load it unpacked for development
2. Sign in to Lope at [app.withlope.com](https://app.withlope.com)
3. The extension uses your Lope session for authentication

### How to add a candidate from LinkedIn

1. Open a LinkedIn profile (or LinkedIn Recruiter search results)
2. Open the Lope side panel (click the extension icon or use the extension's toggle)
3. The extension shows the profile and whether it already exists in Lope
4. Select the **client** and **job** (optional) from the dropdowns
5. Click **Add Candidate**
6. The candidate is added and LinkedIn enrichment is queued (profile data is fetched automatically)

### Features

- **Profile lookup** — See if a LinkedIn profile already exists in your Lope database
- **Add Candidates** — Add one profile at a time with client/job selection
- **Bulk Add** — Select multiple candidates from LinkedIn Recruiter search results and add them in one go
- **Enrichment** — After adding, Lope fetches profile data (name, experience, skills, etc.) in the background

### Requirements

- You must be signed in to Lope in the same browser
- The extension needs access to `linkedin.com` and `withlope.com`
